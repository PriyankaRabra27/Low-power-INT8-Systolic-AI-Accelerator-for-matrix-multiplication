# Low-power-INT8-Systolic-AI-Accelerator-for-matrix-multiplication
This project implements a Low-Power INT8 Systolic Array AI Accelerator in Verilog for matrix multiplication. The design is based on multiply-accumulate Processing Elements that support signed INT8 inputs and INT32 accumulation. Low-power RTL techniques such as zero-skipping, operand isolation, and enable-based register updates are included to reduce unnecessary switching activity. The project is verified using testbenches in Vivado.
